FLOW:
-----
test_job.py (Python)
    ↓ enqueue RQ job
Redis Queue
    ↓ listen & spawn container
main.py (Orchestrator)
    ├── mount: /snap/chromium/current (from host)
    ├── mount: /app/Proxy (from host)
    └── spawn: worker-fnac container
        ├── run.py (entry point)
        ├── sourceCode/main.py
        ├── sourceCode/extractor.py
        └── result → Redis

CONFIG HIERARCHY:
-----------------
.env              (user-editable, docker-compose loads)
    ↓
REDIS_HOST, REDIS_PORT, etc. (env vars in container)
    ↓
config.py         (Python code reads os.environ.get())
    ↓
main.py, run.py   (use config values)
